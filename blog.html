Frequently Asked Questions

What is the Document Object Model (DOM)?

The DOM represents and manipulates HTML documents as objects.

How do we select an element from DOM?

Using getElementById, querySelector, or similar DOM selection methods.

What is the event bubbling in DOM?

Event bubbling propagates events from target element to parent elements.

Why event delegation is important and how it works?

Event delegation improves performance by attaching a single listener to a parent, handling events from child elements dynamically.
